# popusintes-rack

Popusintes es una línea de sintetizadores desarrollada por piruetas.

Este repositorio aloja sus versiones rack, para ser usadas en el software VCV Rack.

## Desarrollo

Módulos programados en C++, usando el editor de texto Visual Studio Code.

## Compilar

```bash
make dep
make
make dist
```

## Gráficas

Las gráficas están hechas con el software Inkscape, y en formato SVG.

Los textos usan la tipografía Talagante de Uno Cá.
